Inspiration and ideas taken from Different Websites

Website 1: Broke By Books

This research page showcases the ideas and inspiration taken from different websites and how they were incorporated into this website. In order to develop this site, references were drawn from a variety of good websites. So, this page is dedicated to displaying and contrasting the components used on our website against the website that is investigated for research purposes.

The first reference taken is from a book blog website created by Sarah S. Davis called Broke By Books and it is relatively simple. On the home page of the website, a big and bold welcome message is displayed which has a good first impression. It gives a warm and welcoming feeling and so it was implemented in our website.

As show in the images, a similar message is displayed on our website but, it contrasts from the reference website as custom font is used and the font colors used are according to the color pallet of our website.

Website 2: Goodreads

Here reference taken is from Goodreads. It is the worldâ€™s largest site for readers and book recommendations. The footer of this website contains links to their social media platforms, links to different pages, download links for their app and copyright message. So, it seemed to be an important element of the website and thus was implement on our website.

In the images shown above the footers of both websites are highlighted in a red rectangular box. "hr" tag is used to create a separating line for the copyright message at the bottom.

Website 3: Booksmandala

The reference taken is from Booksmandal's website. Booksmandal is an online bookstore that also has a physical store in Pokhara, Nepal. This website has an image slider on the top section of the home page that shows attractive offers on different new books and old books. The image slider also has a previous and next button which help to navigation between the images.

In the image shown on the left hand side, the image slider of the reference website is highlighted in a red rectangular box. In the image shown in the right hand side, the image slider of our website is displayed. In our website, the image slider is used to display the genres of books displayed in the products page. We can go to the desired genre just by clicking on the image of the respective genre. Instead of the previous and next button, radio buttons have been used for each image in the slider so that we can navigated to any image at once. This was done using JavaScript and CSS.

Website 4: Fcbarcelona

The reference taken is from Fcbarcelona's website. Fcbarcelona is a very famous football club situated in the city of Barcelona. On their website, they have used hover effect in various places to make the website more interactive and attractive. In the "Only For Culers" section of the home page the categories pop up as we hover over them with our cursor. This caught my eye as it seemed interactive and in a way improved the user experience.

In the image shown on the left hand side, the hover effect of the reference website can be seen and it creates an effect of enlarging the image as we hover over it. In the image shown on the right hand side, the hover effect used in the products page of our website is shown. In our website, the hover effect moves the image of the book 10 pixels upwards as we hover our cursor over it. This was done using the ":hover" selector in CSS.

Website 5: Sara Dietschy

The reference taken is from Sara Dietschy's personal profile website. Sara Dietschy is a youtuber and content creator based in Dallas, Tx, USA. On her website, a form is used in the Contact page for users to sign up to her email for news and updates. This is an important feature for a website so, it is incorporated in our website.

In the image show on the left hand side, the contact page of the reference website is shown. The first name, last name and email of the user are taken as input in her website and there is a "Sign Up" button for signing up to her email services. A thank you message is displayed if correct data is filled on clicking the "Sign Up" button. In the image show on the right hand side, the Contact Us page of our website is shown. In our website, the form in the Contact Us page is uesd to receive messages of feedback and queries form the users. The first name, last name, email of the user and their message is taken as input in our website and there is a "Send Message" button for sending the message to us. If any field is empty or email address is invalid a suitable error message is displayed and if all data entries are vaild then a "Message Sent!" text is displayed. Here, external JavaScript is used for form validation.
